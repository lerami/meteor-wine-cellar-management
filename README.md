## Getting Started

Clone the project:
```
git clone https://github.com/lerami/meteor-wine-cellar-management.git
```

Run it with meteor:
```
cd meteor-wine-cellar-management
meteor
```

### Prerequisites

You'll need to install [Meteor](https://www.meteor.com/) first if not already installed on your machine.
